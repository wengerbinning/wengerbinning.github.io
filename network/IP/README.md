### 有类分类地址

| 类别 | 前缀标识 | 
|:---- |:------ |
| A类 | 0b0     | 
| B类 | 0b10    | 
| C类 | 0b110   |
| D类 | 0b1110  |
| E类 | 0b11110 |


#### A类

在32位地址中， 前一位为0的为A类地址：总共约有21亿个地址；

