




#### subst

用于在字符串中替换子串。

```make
$(subst from, to, text)
```

#### patsubst

用于使用正则来替换字符串中的子串。

```make
$(patsubst pattern, replacement, text)
```

#### strip

移除字符串前后的空白字符

```make
$(strip string)
```
