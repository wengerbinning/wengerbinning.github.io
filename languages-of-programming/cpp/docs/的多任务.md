## 进程

进程（Process）是计算机的运行的程序。在早期的分时系统中，进程作为最基本的运作单位；在早期以进程为主体的系统中（Linux 2.4及之前），进程是程序的基本执行实体；在现今以线程为主题的系统中（Linux 2.6及之后），进程是线程的容器。进程拥有新建、运行、等待、就绪、结束5种状态。

* 在Linux下创建子进程的举例：

  ```c++
  #include <iostream>
  #include <sys/types.h>
  #include <unistd.h>

  int main() {
      pid_t pid = fork();
  }
  ```

  > Note：创建的子进程执行同样的程序。

* 在Windows下创建子进程的举例：在Windows中，使用`CreateProcess()`等创建子进程。
