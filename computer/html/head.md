## 元素列表

在head中可以使用以下元素。

### title

title标签说明文档的标题。

### meta

meta标签说明文档一些属性， 例如编码、作者等。是一个空元素。

```html
<meta charset="utf-8"/>
```

### base

base标签描述了基本的链接地址/链接目标，该标签作为HTML文档中所有的链接标签的默

### style

style标签标记一段CSS样式。

### link

link标签标记了文档与外部资源的关系， 一般用来链接样式表。
