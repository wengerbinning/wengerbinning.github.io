
#### MAC Sublayer Frame

| size | Label |
|:----:|:----- |
| 2B      | Frame Control        |
| 2B      | Duration             |
| 6B      | Address1             |
| 6B      | Address2             |
| 6B      | Address3             |
| 2B      | Sequence             |
| 6B      | *Address4             |
| 0~2312B | Data                 |
| 4B      | Frame Check Sequence |

| 1b   | FromDS           |
| 1b   | More Fragments   |
| 1b   | Retry            |
| 1b   | Power Management |
| 1b   | More Data        |
| 1b   | Protected Frame  |
| 1b   | Order            |
