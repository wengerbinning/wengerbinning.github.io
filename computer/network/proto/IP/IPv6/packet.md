## IPv6 Header


* 004b - Version
* 008b - Traffic Class
* 020b - Flow Label

* 016b - Playload length
* 008b - Next Header
* 008b - Hop Limit

* 128b - source address
* 128b - destination address

## IPv6 Option

* 000 - Hop-by-Hop options
* 043 - Routing
* 044 - Fragment
* 051 - Authentication Header(AH)
* 050 - ESP
* 060 - Destination options
