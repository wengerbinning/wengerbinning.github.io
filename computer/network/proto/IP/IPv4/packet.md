

## IPv4 Header

* 04b - Version
* 04b - IHL(IP Header Length)
* 08b - TOS(Type of Service)
* 16b - Total Length

* 16b - Identification
* 03b - Flags
* 13b - Fragment offset

* 08b - TTL
* 08b - Protocol
* 16b - Header Checksum

* 32b - source ip addr
* 32b - destination ip addr

## IPv4 Option

* xxb - option(maxum 40B)

## IPv4 payload

* xxb - playload
