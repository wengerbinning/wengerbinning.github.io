## IKEv1

* 第一阶段，通信双方协商并建立IKE SA，该阶段支持两种模式

  1. Main Mode (主模式)
  2. Aggressive Mode (野蛮模式)

* 第二阶段，基于第一阶段已通过认证的安全通道，建立一条数据安全传输通道IPsec SA。
