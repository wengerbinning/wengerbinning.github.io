class 是实现一个数据类型的， 通常一个文件内实现一个类，类名与文件名称保持一致。


```java
public class Main {
  int x = 5;

  public static void main(String[] args) {
    Main myObj = new Main();
    System.out.println(myObj.x);
  }
}
```
