


#### Element Selectors

Simple Selector

* 标签选择器(name selector)

```css
h1 {
    color: white;
}
```

* 类型选择器(class selector)

```css
.account {
    color: black;
}
```

* ID选择器(id selector)

```css
#paaword {
    color: yellow;
}
```
