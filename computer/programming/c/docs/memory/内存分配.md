


## std-libc

#### malloc

```c
void * malloc (size_ t size);
```

申请一块连续的内存；

* 返回申请内存的首地址;
* size指定申请内存的大小;
* 申请失败后返回NULL;
* 申请内存后不做初始化;

### calloc

