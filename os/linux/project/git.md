








## tag

* 创建TAG

```shell

git tag 

git tag -a
git tag -m

```


* 查看TAG

```shell

git tag

git tag --sort=committerdate

git tag -l "v*"


```

*  删除标签

```shell
git tag -d
```



## archive

源码打包， 

```shell

git archive master --format=tar.gz --output=../master.tar.gz


```
