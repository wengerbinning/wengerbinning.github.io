# Email --电子邮件与MIME处理包
```
author: ClarkAaron
creationTime: 2020-01-29 19:18
```
* Email是Python管理邮件信息的内置标准库,它并不是为发送给邮件而设计的,email尝试满足RFC,支持RFC5233与RFC6532等;
* 分别由"对象模型(Object model,a tree structure of objects)"、"解析器(Parer)"、"生成器(generator)"与"控制(control)"四大组件组成;
