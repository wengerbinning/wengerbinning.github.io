Latex是一个文档排版系统。


* 所有命令以`\`标识
* 所有注释以`%`标识

* 环境(代码块)：公式环境、数据环境、章节环境，以`\begin{}`开始, 以`\end{}`结尾；
* Latex源码分为导言区与正文区
* 导言区`\documentclass{ctexart}`开始，到正文区结束， 用于引入各种宏包；
* 正文区是在document环境中(`\begin{document}`与`\end{document}`)，一份文件中只能存在一个正文区；
